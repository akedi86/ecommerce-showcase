# Akedi Store – E‑Commerce Showcase

A modern, responsive e‑commerce showcase built with **React** and **Tailwind CSS**, deployed on **Vercel**.  
This project demonstrates frontend development, clean UI design, and deployment skills 

---

##  Features
-  Product grid with sample DevOps/cloud‑themed items
-  Styled using Tailwind CSS for modern, responsive design
-  Fast deployment on Vercel with CI/CD support
-  Mobile‑friendly layout

---

##  Tech Stack
- **React** (frontend framework)
- **Tailwind CSS** (styling)
- **Vercel** (deployment)
- **GitHub Actions** (CI/CD)

---

```bash
git clone https://github.com/YOURUSERNAME/ecommerce-showcase.git
cd ecommerce-showcase
